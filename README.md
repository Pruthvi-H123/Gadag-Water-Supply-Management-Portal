# 💧 Gadag Water Supply Management Portal

A web-based Water Supply Management System developed using **Python Flask** and **MySQL** to simplify the management of water supply services. The system allows users to register, apply for new water connections, pay bills, submit complaints, and track requests, while providing administrators with tools to manage consumers, billing, and service requests efficiently.

## 🚀 Features

### User Module
- User Registration & Login
- Dashboard
- Apply for New Water Connection
- View Connection Status
- Online Bill Payment
- View Payment History
- Submit Complaints
- Track Complaint Status
- Profile Management

### Admin Module
- Secure Admin Login
- Manage Users
- Approve/Reject New Connection Requests
- Generate Water Bills
- Manage Payments
- View and Resolve Complaints
- Dashboard with System Overview

## 🛠️ Technologies Used

- **Frontend:** HTML, CSS, JavaScript, Bootstrap
- **Backend:** Python Flask
- **Database:** MySQL
- **Tools:** VS Code, XAMPP


## ⚙️ Installation

1. Clone the repository
```bash
git clone https://github.com/your-username/Gadag-Water-Supply-Management-Portal.git
```

2. Navigate to the project folder
```bash
cd Gadag-Water-Supply-Management-Portal
```

3. Install dependencies
```bash
pip install -r requirements.txt
```

4. Create a MySQL database and import the SQL file.

5. Update database credentials in `app.py`.

6. Run the application
```bash
python app.py
```

7. Open your browser and visit
```
http://127.0.0.1:5000/
```
